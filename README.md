Diese Website täuscht vor, nach Beiträgen zum Thema "Weltraumtourismus" zu suchen.
In Wirklichkeit wurden die Beiträge bereits im Vorhinein so ausgewählt, dass sie Weltraumtourismus möglichst positiv darstellen.
Die ausgewählten Beiträge sind in ContentData.json zu finden.
Über die Chat-App kann mit Hilfe von Google Gemini über Weltraumtourismus gesprochen werden, wobei der Chatbot hier die Vorteile von Weltraumtourismus mehr betont als dessen Nachteile.
Eine fast identische Website, die sich auf die negativen Aspekte von Weltraumtourismus fokussiert, ist hier zu finden: https://recherchehilfe.github.io/Home/

Die zwei Websites können z.B. im Unterricht verwendet werden, um den Schüler:innen klarzumachen, wie leicht Filterblasen die eigene Meinung beeinflussen können und um ihre kritische Medienkompetenz zu steigern.
